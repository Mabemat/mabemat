      Chapitre 07 - Leçon

      Calcul littéral (2)

      7.1 Identités remarquables

      7.1.1 Développer avec une identité remarquable

      Définition :
      On considère $a, b, c, d$ quatre nombres.
      • $(a + b)^2 = a^2 + 2ab + b^2$
      • $(a - b)^2 = a^2 - 2ab + b^2$
      • $(a - b)(a + b) = a^2 - b^2$
      Exemples
      Développer les expressions suivantes.
      • $(x + 7)^2 = x^2 + 2 \times x \times 7 + 7^2 = x^2 + 14x + 49$
      • $(3x - 5)^2 = (3x)^2 - 2 \times 3x \times 5 + 5^2 = 9x^2 - 30x + 25$
      • $(5x - 6)(5x + 6) = (5x)^2 - 6^2 = 25x^2 - 36$

      7.1.2 Factoriser avec une identité remarquable

      Exemples
      Factoriser ces expressions à l'aide d'une identité remarquable.
      • $x^2 + 10x + 25 = x^2 + 2 \times x \times 5 + 5^2 = (x + 5)^2$
      • $4x^2 - 12x + 9 = (2x)^2 - 2 \times 2x \times 3 + 3^2 = (2x - 3)^2$
      • $x^2 - 4 = (x - 2)(x + 2)$
      • $9x^2 - 16 = (3x)^2 - 4^2 = (3x - 4)(3x + 4)$

      7.2 Équation produit

      Propriété :
      Un produit de facteur est nul si et seulement si l'un des facteurs est nul. Autrement dit,
      • Si $A \times B = 0$, alors $A = 0$ ou $B = 0$.
      • Si $A = 0$ ou $B = 0$, alors $A \times B = 0$.
      Exemples
      Résoudre $(x + 2)(x - 3) = 0$.
      $$ \begin{align*} (x + 2)(x - 3) &= 0 \\ x + 2 = 0 \quad &\text{ou} \quad x - 3 = 0 \\ x = -2 \quad &\text{ou} \quad x = 3 \end{align*} $$

      Vérification :

      • Si $x = -2$, alors $(x + 2)(x - 3) = (-2 + 2) \times (-2 - 3) = 0 \times -5 = 0$.
      • Si $x = 3$, alors $(x + 2)(x - 3) = (3 + 2) \times (3 - 3) = 5 \times 0 = 0$.
      Quand le produit n'est pas mis en évidence, on factorise.
      Exemples
      $$ \begin{align*} 2x^2 + 15x &= 0 \\ 2x \times x + 15 \times x &= 0 \\ x(2x + 15) &= 0 \\ x = 0 \quad \text{ou} \quad &2x + 15 = 0 \\ x = 0 \quad \text{ou} \quad &x = -\dfrac{15}{2} \end{align*} $$
      $$ \begin{align*} x^2 - 36 &= 0 \\ x^2 - 6^2 &= 0 \\ (x + 6)(x - 6) &= 0 \\ x + 6 = 0 \quad \text{ou} \quad &x - 6 = 0 \\ x = -6 \quad \text{ou} \quad &x = 6 \end{align*} $$
      $$ \begin{align*} x^2 - 3 &= 0 \\ x^2 - \sqrt{3}^2 &= 0 \\ (x + \sqrt{3})(x - \sqrt{3}) &= 0 \\ x + \sqrt{3} = 0 \quad \text{ou} \quad &x - \sqrt{3} = 0 \\ x = -\sqrt{3} \quad \text{ou} \quad &x = \sqrt{3} \end{align*} $$

      7.3 Équation du type $ x^2=a $

      Propriété :
      Pour tout nombre $a$,
      • si $a < 0$ alors l'équation $x^2 = a$ n'a pas de solution ;
      • si $a = 0$ alors l'équation $x^2 = 0$ a une solution $0$.
      Exemple
      L'équation $x^2 = -8$ n'a pas de solution car $-8 \leq 0$.
      Propriété :
      Pour tout nombre $a$ positif, l'équation $x^2 = a$ a deux solutions : $\sqrt{a}$ et $-\sqrt{a}$.
      Exemples
      • L'équation $x^2 = 5$ a deux solutions : $\sqrt{5}$ et $-\sqrt{5}$.
      • L'équation $x^2 = 36$ a deux solutions : $6$ et $-6$.
      Exemples
      $$ \begin{align*} -2x^2&=-4 \\ \dfrac{-2x^2}{-2}&=\dfrac{-4}{-2} \\ x^2&=2 \\ x=\sqrt{2} &\text{ ou } x=-\sqrt{2} \end{align*} $$
      $$ \begin{align*} 16x^2 -81&=0 \\ 16x^2 &= 81 \\ x^2 &= \frac{81}{16} \\ x = \sqrt{\frac{81}{16}} &\text{ ou } x = -\sqrt{\frac{81}{16}} \\ x = \frac{9}{4} &\text{ ou } x = -\frac{9}{4} \end{align*} $$
