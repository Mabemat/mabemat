QCM — Angles

Quel est le nom de l’angle formé par deux demi-droites qui forment une droite ?
Angle nul
Angle plein
Angle droit
Angle plat
Quand deux demi-droites de même origine forment une droite, l’angle obtenu mesure 180° : c’est un angle plat.
Un angle aigu mesure :
Entre 0° et 90° (exclu)
Exactement 90°
Entre 90° et 180°
Exactement 180°
  • Angle aigu : entre 0° et 90° (strictement)
  • Angle droit : exactement 90°
  • Angle obtus : entre 90° et 180° (strictement)
  • Angle plat : exactement 180°
Deux angles sont supplémentaires. L’un mesure 48°. Quelle est la mesure de l’autre ?
25°
48°
132°
Aucune réponse

Deux angles supplémentaires ont une somme de 180° :

$$180° - 48° = \mathbf{132°}$$
Deux droites se coupent. L’un des angles formés mesure 38°. Quelle est la mesure de l’angle opposé par le sommet ?

130°
90°

Les angles opposés par le sommet sont toujours égaux.

L’angle opposé par le sommet mesure donc aussi 38°.

Comment peut-on nommer l’angle gris dans la figure ci-dessous ?

$\widehat{MRI}$
$\widehat{RIM}$
$\widehat{IMR}$
Aucune réponse

Un angle se nomme avec trois lettres : le sommet est la lettre du milieu. Ici le sommet est $R$, et les deux côtés passent par $M$ (ou $K$) et $I$ (ou $U$).

$\widehat{MRI}$ est donc un nom valide (parmi d’autres : $\widehat{IRM}$, $\widehat{MRU}$, $\widehat{KRI}$…).

Dans la figure ci-dessous, $\widehat{XEJ}$ est :

le sommet de l’angle
un côté de l’angle
le nom de l’angle
rien de cela

$\widehat{XEJ}$ n’est pas le nom de l’angle : le sommet serait $E$, mais l’angle formé par les demi-droites $[EX)$ et $[EJ)$ se noterait $\widehat{XEJ}$… sauf que le sommet ici est $X$, pas $E$. C’est donc rien de cela.

Le nom correct de l’angle est $\widehat{JXE}$ (sommet $X$).

Dans la même figure, $X$ est :

le sommet de l’angle
un côté de l’angle
le nom de l’angle
rien de cela
Le sommet d’un angle est l’origine commune des deux demi-droites qui le forment. Ici les deux demi-droites $[XJ)$ et $[XE)$ partent de $X$ : $X$ est donc le sommet.

Dans la même figure, $[XE)$ est :

le sommet de l’angle
un côté de l’angle
le nom de l’angle
rien de cela
Les côtés d’un angle sont les deux demi-droites qui le forment. $[XE)$ est l’une des deux demi-droites d’origine $X$ : c’est donc un côté de l’angle.

Dans la même figure, $\widehat{JXE}$ est :

le sommet de l’angle
un côté de l’angle
le nom de l’angle
rien de cela
$\widehat{JXE}$ est le nom de l’angle : la lettre centrale $X$ est le sommet, et les lettres $J$ et $E$ désignent un point sur chaque côté. C’est la notation correcte.

aigu

Aucune réponse
Un angle obtus mesure entre $90°$ et $180°$. L’arc de l’angle $\widehat{JCL}$ est clairement plus grand qu’un angle droit : c’est un angle obtus.

Estimer la mesure de l’angle $\widehat{tLs}$ sans instrument.

$10°$
$100°$
$145°$
$55°$
$90°$
Aucune réponse

Les deux demi-droites $[Lt)$ et $[Ls)$ sont presque confondues — l’angle entre elles est très faible. C’est un angle très petit, d’environ $\mathbf{10°}$.

Les autres valeurs (90°, 145°…) correspondrait à des angles beaucoup plus ouverts.

La demi-droite $[WE)$ est la bissectrice de l’angle $\widehat{DWB}$, et $\widehat{DWB} = 72°$. Quelle est la mesure de l’angle $\widehat{DWE}$ ?
$72°$
$36°$
$136°$
Aucune réponse

Par définition, la bissectrice d’un angle le partage en deux angles de même mesure.

$$\widehat{DWE} = \frac{\widehat{DWB}}{2} = \frac{72°}{2} = \mathbf{36°}$$
