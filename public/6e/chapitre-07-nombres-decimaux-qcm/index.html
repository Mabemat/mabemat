QCM — Nombres décimaux

Quelle est l’écriture décimale de $\dfrac{6375}{1000}$ ?
$0{,}6375$
$63{,}75$
$6{,}375$
Aucune réponse

On divise par $1000$, ce qui revient à déplacer la virgule de trois rangs vers la gauche :

$$\dfrac{6375}{1000} = 6{,}375.$$
Laquelle de ces fractions est une fraction décimale ?
$\dfrac{7}{3}$
$\dfrac{14}{100}$
$\dfrac{5}{12}$
Aucune réponse

Une fraction décimale est une fraction dont le dénominateur est $10$, $100$, $1\,000$, etc.

Ici, seule : $\dfrac{14}{100}$ a un dénominateur décimal.

Dans le nombre $851{,}67$, quel est le chiffre des dixièmes ?
$7$
$6$
$5$
Aucune réponse

Dans $851{,}67$ :

  • $8$ est le chiffre des centaines,
  • $5$ celui des dizaines,
  • $1$ celui des unités,
  • $6$ celui des dixièmes,
  • $7$ celui des centièmes.

Le chiffre des dixièmes est donc $6$.

Dans le nombre $851{,}67$, quel est le nombre de dizaines ?
$6$
$5$
$85$
Aucune réponse

Le nombre de dizaines, c’est le quotient de $851{,}67$ par $10$ pris entier :

$$851{,}67 = 85 \times 10 + 1{,}67.$$

Le nombre de dizaines est donc $85$ (à ne pas confondre avec le chiffre des dizaines, qui est $5$).

Quel est le nombre correspondant à 347 unités, 9 dixièmes et 82 millièmes ?
$347{,}892$
$347{,}982$
$347{,}829$
Aucune réponse

On place chaque partie dans la bonne colonne :

Centaines Dizaines Unités Dixièmes Centièmes Millièmes
3 4 7
0, 9
0, 0 8 2
3 4 7, 9 8 2

Donc : $347 + 0{,}9 + 0{,}082 = 347{,}982$
