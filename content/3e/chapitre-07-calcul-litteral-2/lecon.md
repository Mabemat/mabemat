<h2 class="subsection-title">7.1 Identités remarquables</h2>

<h3 class="subsubsection-title">7.1.1 Développer avec une identité remarquable</h3>

<div class="environment definition">
<div class="environment-title">Définition :</div>
<div class="environment-content">
On considère $a, b, c, d$ quatre nombres.

<ul style="display:grid; grid-template-columns: repeat(3, 1fr); gap:2rem;">
<li>$(a + b)^2 = a^2 + 2ab + b^2$</li>
<li>$(a - b)^2 = a^2 - 2ab + b^2$</li>
<li>$(a - b)(a + b) = a^2 - b^2$</li>
</ul>

</div>
</div>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">
Développer les expressions suivantes.

<ul>
<li>$(x + 7)^2 = x^2 + 2 \times x \times 7 + 7^2 = x^2 + 14x + 49$</li>
<li>$(3x - 5)^2 = (3x)^2 - 2 \times 3x \times 5 + 5^2 = 9x^2 - 30x + 25$</li>
<li>$(5x - 6)(5x + 6) = (5x)^2 - 6^2 = 25x^2 - 36$</li>
</ul>
</div>
</div>

<h3 class="subsubsection-title">7.1.2 Factoriser avec une identité remarquable</h3>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">
Factoriser ces expressions à l'aide d'une identité remarquable.

<ul>
<li>$x^2 + 10x + 25 = x^2 + 2 \times x \times 5 + 5^2 = (x + 5)^2$</li>
<li>$4x^2 - 12x + 9 = (2x)^2 - 2 \times 2x \times 3 + 3^2 = (2x - 3)^2$</li>
<li>$x^2 - 4 = (x - 2)(x + 2)$</li>
<li>$9x^2 - 16 = (3x)^2 - 4^2 = (3x - 4)(3x + 4)$</li>
</ul>
</div>
</div>

<h2 class="subsection-title">7.2 Équation produit</h2>

<div class="environment propriete">
<div class="environment-title">Propriété :</div>
<div class="environment-content">
Un produit de facteur est nul si et seulement si l'un des facteurs est nul. Autrement dit,

<ul style="column-count:2; column-gap:2rem;">
<li>Si $A \times B = 0$, alors $A = 0$ ou $B = 0$.</li>
<li>Si $A = 0$ ou $B = 0$, alors $A \times B = 0$.</li>
</ul>

</div>
</div>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">
Résoudre $(x + 2)(x - 3) = 0$.

<div class="math-display">
$$
\begin{align*}
(x + 2)(x - 3) &= 0 \\
x + 2 = 0 \quad &\text{ou} \quad x - 3 = 0 \\
x = -2 \quad &\text{ou} \quad x = 3
\end{align*}
$$
</div>

<strong>Vérification :</strong>
<ul>
<li>Si $x = -2$, alors $(x + 2)(x - 3) = (-2 + 2) \times (-2 - 3) = 0 \times -5 = 0$.</li>
<li>Si $x = 3$, alors $(x + 2)(x - 3) = (3 + 2) \times (3 - 3) = 5 \times 0 = 0$.</li>
</ul>
</div>
</div>

<div class="remarque">
Quand le produit n'est pas mis en évidence, on factorise.
</div>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">

<div class="math-display" style="display:inline-block; vertical-align:top; width:calc(32% - 10px); margin:0 5px 5px 0;">
$$
\begin{align*}
2x^2 + 15x &= 0 \\
2x \times x + 15 \times x &= 0 \\
x(2x + 15) &= 0 \\
x = 0 \quad \text{ou} \quad &2x + 15 = 0 \\
x = 0 \quad \text{ou} \quad &x = -\dfrac{15}{2}
\end{align*}
$$
</div><div class="math-display" style="display:inline-block; vertical-align:top; width:calc(32% - 10px); margin:0 5px 5px 0;">
$$
\begin{align*}
x^2 - 36 &= 0 \\
x^2 - 6^2 &= 0 \\
(x + 6)(x - 6) &= 0 \\
x + 6 = 0 \quad \text{ou} \quad &x - 6 = 0 \\
x = -6 \quad \text{ou} \quad &x = 6
\end{align*}
$$
</div><div class="math-display" style="display:inline-block; vertical-align:top; width:calc(32% - 10px); margin:0 5px 5px 0;">
$$
\begin{align*}
x^2 - 3 &= 0 \\
x^2 - \sqrt{3}^2 &= 0 \\
(x + \sqrt{3})(x - \sqrt{3}) &= 0 \\
x + \sqrt{3} = 0 \quad \text{ou} \quad &x - \sqrt{3} = 0 \\
x = -\sqrt{3} \quad \text{ou} \quad &x = \sqrt{3}
\end{align*}
$$
</div>

</div>
</div>

<h2 class="subsection-title">7.3 Équation du type $ x^2=a $</h2>

<div class="environment propriete">
<div class="environment-title">Propriété :</div>
<div class="environment-content">
Pour tout nombre $a$,
<ul>
<li>si $a < 0$ alors l'équation $x^2 = a$ n'a pas de solution ;</li>
<li>si $a = 0$ alors l'équation $x^2 = 0$ a une solution $0$.</li>
</ul>
</div>
</div>

<div class="environment exemple">
<div class="environment-title">Exemple</div>
<div class="environment-content">
L'équation $x^2 = -8$ n'a pas de solution car $-8 \leq 0$.
</div>
</div>

<div class="environment propriete">
<div class="environment-title">Propriété :</div>
<div class="environment-content">
Pour tout nombre $a$ positif, l'équation $x^2 = a$ a deux solutions : $\sqrt{a}$ et $-\sqrt{a}$.
</div>
</div>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">

<ul>
<li>L'équation $x^2 = 5$ a deux solutions : $\sqrt{5}$ et $-\sqrt{5}$.</li>
<li>L'équation $x^2 = 36$ a deux solutions : $6$ et $-6$.</li>
</ul>
</div>
</div>

<div class="environment exemple">
<div class="environment-title">Exemples</div>
<div class="environment-content">

<div class="math-display" style="display:inline-block; vertical-align:top; width:calc(49% - 10px); margin:0 5px 5px 0;">
$$
\begin{align*}
-2x^2&=-4 \\
\dfrac{-2x^2}{-2}&=\dfrac{-4}{-2} \\
x^2&=2 \\
x=\sqrt{2} &\text{ ou } x=-\sqrt{2}
\end{align*}
$$
</div><div class="math-display" style="display:inline-block; vertical-align:top; width:calc(49% - 10px); margin:0 5px 5px 0;">
$$
\begin{align*}
16x^2 -81&=0  \\
16x^2 &= 81  \\
x^2 &= \frac{81}{16}  \\
x = \sqrt{\frac{81}{16}} &\text{ ou } x = -\sqrt{\frac{81}{16}}  \\
x = \frac{9}{4} &\text{ ou } x = -\frac{9}{4} 
\end{align*}
$$
</div>

</div>
</div>
