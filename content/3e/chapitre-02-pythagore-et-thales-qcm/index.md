{{< qcm question="Si $(AC)^2 = (AB)^2 + (BC)^2$, alors le triangle $ABC$ est :" correct="0" >}}
- Rectangle en $B$
- Isocèle
- Equilatéral
- Quelconque
{{< /qcm >}}

{{< qcm question="Le théorème de Thalès s'applique quand :" correct="1" >}}
- Deux angles sont égaux
- Deux droites sont parallèles
- Deux côtés sont égaux
- Un angle est droit
{{< /qcm >}}

{{< qcm question="Dans un triangle $ABC$ avec $D$ sur $[AB]$ et $E$ sur $[AC]$ tels que $(DE) \parallel (BC)$, alors :" correct="0" >}}
- $\frac{AD}{AB} = \frac{AE}{AC}$
- $\frac{AB}{AD} = \frac{BC}{DE}$
- $AD = AE$
- $\frac{AB}{DB} = \frac{AE}{EC}$
{{< /qcm >}}

{{< qcm question="La réciproque du théorème de Pythagore permet de :" correct="2" >}}
- Calculer une aire
- Trouver un angle
- Vérifier si un triangle est rectangle
- Trouver une hypoténuse
{{< /qcm >}}

{{< qcm question="Dans un triangle rectangle, l'hypoténuse est :" correct="1" >}}
- Le côté le plus court
- Le côté opposé à l'angle droit
- N'importe quel côté
- Le côté adjacent à l'angle droit
{{< /qcm >}}
