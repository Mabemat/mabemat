{{< qcm question="Une fonction est une relation qui à chaque valeur $x$ associe :" correct="2" >}}
- Plusieurs valeurs
- Aucune valeur
- Une unique valeur $f(x)$
- Toujours la même valeur
{{< /qcm >}}

{{< qcm question="L'image de $3$ par $f(x) = 2x - 1$ est :" correct="0" >}}
- $5$
- $4$
- $6$
- $7$
{{< /qcm >}}

{{< qcm question="L'antécédent de $7$ par $f(x) = 3x + 1$ est :" correct="1" >}}
- $3$
- $2$
- $4$
- $1$
{{< /qcm >}}

{{< qcm question="Le domaine de définition d'une fonction est :" correct="1" >}}
- L'ensemble des images
- L'ensemble des valeurs pour lesquelles $f$ est définie
- L'ensemble des antécédents positifs
- Toujours $\mathbb{R}$
{{< /qcm >}}

{{< qcm question="Si $f(2) = 4$ et $f(3) = 9$, une fonction possible est :" correct="2" >}}
- $f(x) = 2x$
- $f(x) = x + 2$
- $f(x) = x^2$
- $f(x) = 3x - 2$
{{< /qcm >}}
