{{< lycee-exo num="1" source="Supprimer les parenthèses et réduire les expressions suivantes." >}}
---INTRO---
---Q---
$A= (-6b^2+9b-2) - (-4b^2+5b+8)$
---CORR---
$A= (-6b^2+9b-2) - (-4b^2+5b+8)$<br>$A= -6b^2+9b-2+4b^2-5b-8$<br>$A=$ ${\color{#C5607A}\boldsymbol{-2b^2+4b-10}}$
---Q---
$B= (-3b-6) - (-5b^2+b+7)$
---CORR---
$B= (-3b-6) - (-5b^2+b+7)$<br>$B= -3b-6 +5b^2-b-7$<br>$B=$ ${\color{#C5607A}\boldsymbol{5b^2-4b-13}}$
---Q---
$C= -(-5a^2+a-5) + (8a^2+2a+4)$
---CORR---
$C= -(-5a^2+a-5) + (8a^2+2a+4)$<br>$C= 5a^2-a+5 +8a^2+2a+4$<br>$C=$ ${\color{#C5607A}\boldsymbol{13a^2+a+9}}$
---Q---
$D= (5b^2-5b-5) - (-6b^2+9b-10)$
---CORR---
$D= (5b^2-5b-5) - (-6b^2+9b-10)$<br>$D= 5b^2-5b-5+6b^2-9b+10$<br>$D=$ ${\color{#C5607A}\boldsymbol{11b^2-14b+5}}$
---Q---
$E= -(-4y-5) + (-2y^2+3y+1)$
---CORR---
$E= -(-4y-5) + (-2y^2+3y+1)$<br>$E= 4y+5 -2y^2+3y+1$<br>$E=$ ${\color{#C5607A}\boldsymbol{-2y^2+7y+6}}$
---Q---
$F= -(8a+7) + (7a+3)$
---CORR---
$F= -(8a+7) + (7a+3)$<br>$F= -8a-7+7a+3$<br>$F=$ ${\color{#C5607A}\boldsymbol{-a-4}}$
{{< /lycee-exo >}}

{{< lycee-exo num="2" source="Développer et réduire les expressions suivantes." >}}
---INTRO---
---Q---
$A=-3-10(-4t+2)$
---CORR---
$A=-3{\color{#8B3C52}\boldsymbol{-10}}(-4t+2)$<br>$A=-3{\color{#8B3C52}\boldsymbol{-10}}\times (-4t)+{\color{#8B3C52}\boldsymbol{(-10)}}\times 2$<br>En réduisant l'expression, on obtient : <br> $A=$ ${\color{#C5607A}\boldsymbol{40t-23}}$.
---Q---
$B=9z(6z-5)$
---CORR---
$B={\color{#8B3C52}\boldsymbol{9z}}(6z-5)$<br>$B={\color{#8B3C52}\boldsymbol{9z}}\times 6z + {\color{#8B3C52}\boldsymbol{9z}}\times (-5)$<br>En réduisant l'expression, on obtient : <br> $B=$ ${\color{#C5607A}\boldsymbol{54z^2-45z}}$.
---Q---
$C=2(8c-3)+6$
---CORR---
$C={\color{#8B3C52}\boldsymbol{2}}(8c-3)+6$<br>$C={\color{#8B3C52}\boldsymbol{2}}\times 8c+{\color{#8B3C52}\boldsymbol{2}}\times (-3)+6$<br>En réduisant l'expression, on obtient : <br> $C=$ ${\color{#C5607A}\boldsymbol{16c}}$.
---Q---
$D=-6(-8x-5)-2$
---CORR---
$D={\color{#8B3C52}\boldsymbol{-6}}(-8x-5)-2$<br>$D={\color{#8B3C52}\boldsymbol{-6}}\times (-8x)+{\color{#8B3C52}\boldsymbol{(-6)}}\times (-5)-2$<br>En réduisant l'expression, on obtient : <br> $D=$ ${\color{#C5607A}\boldsymbol{48x+28}}$.

---INTRO---
---Q---
$A = 3(-9x+4)-8(-8x+2)$
---CORR---
$A =  3(-9x+4)-8(-8x+2)$ <br>$A =  3 \times (-9x) +
      3 \times 4{\color{#8B3C52}\boldsymbol{\,-\Big(}}8 \times (-8x) +
      8 \times 2{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$A = -27x + 
    12{\color{#8B3C52}\boldsymbol{\,-\Big(}}-64x + 
    16{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$A = -27x+12+64x-16$ <br>$A =$ ${\color{#C5607A}\boldsymbol{ 37x-4 }}$
---Q---
$B = -2(3c+6)-(6c-5)(-8c+2)$
---CORR---
$B =  -2(3c+6)-(6c-5)(-8c+2)$ <br>$B =  -2 \times 3c +
      (-2) \times 6{\color{#8B3C52}\boldsymbol{\,-\Big(}}6c\times (-8c) + 
  6c\times 2 +
  (-5) \times (-8c)  +
  (-5) \times 2{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$B = -6c + 
    (-12){\color{#8B3C52}\boldsymbol{\,-\Big(}}-48c^2 + 
  12c +
  40c +
  (-10){\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$B = -6c-12{\color{#8B3C52}\boldsymbol{\,-\Big(}}-48c^2+52c-10{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$B = -6c-12+48c^2-52c+10$ <br>$B =$ ${\color{#C5607A}\boldsymbol{ 48c^2-58c-2 }}$
---Q---
$C = -3(-b+6)+(-8b+6)(2b-5)$
---CORR---
$C =  -3(-b+6)+(-8b+6)(2b-5)$ <br>$C =  -3 \times (-b) +
      (-3) \times 6{\color{#8B3C52}\boldsymbol{\,+\Big(}}-8b\times 2b + 
  (-8b)\times (-5) +
  6 \times 2b  +
  6 \times (-5){\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$C = 3b + 
    (-18){\color{#8B3C52}\boldsymbol{\,+\Big(}}-16b^2 + 
  40b +
  12b +
  (-30){\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$C = 3b-18{\color{#8B3C52}\boldsymbol{\,+\Big(}}-16b^2+52b-30{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$C = 3b-18-16b^2+52b-30$ <br>$C =$ ${\color{#C5607A}\boldsymbol{ -16b^2+55b-48 }}$
---Q---

  (-48n) +
  (-28n) +
  42{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$D = 15n+40{\color{#8B3C52}\boldsymbol{\,-\Big(}}32n^2-76n+42{\color{#8B3C52}\boldsymbol{\Big)}}$ <br>$D = 15n+40-32n^2+76n-42$ <br>$D =$ ${\color{#C5607A}\boldsymbol{ -32n^2+91n-2 }}$

---INTRO---
---Q---
$A=(8b+6)^2$
---CORR---
$A=(8b+6)^2$<br>$A=(8b+6)(8b+6)$<br>$A= 8b \times 8b + 8b \times 6 + 6 \times 8b  + 6 \times 6$<br>$A=64b^2 + 48b + 48b + 36$<br>$A=$ ${\color{#C5607A}\boldsymbol{64b^2+96b+36}}$
---Q---
$B=(4+2c)^2$
---CORR---
$B=(4+2c)^2$<br>$B=(4+2c)(4+2c)$<br>$B= 4 \times 4 +  4 \times 2c +  2c \times 4  + 2c \times 2c $<br>$B=16 + 8c + 8c + 4c^2$<br>$B=$ ${\color{#C5607A}\boldsymbol{4c^2+16c+16}}$
---Q---
$C=(6y-10)^2$
---CORR---
$C=(6y-10)^2$<br>$C=(6y-10)(6y-10)$<br>$C= 6y \times 6y + 6y \times (-10) + (-10) \times 6y  + (-10) \times (-10)$<br>$C=36y^2 - 60y - 60y + 100$<br>$C=$ ${\color{#C5607A}\boldsymbol{36y^2-120y+100}}$
---Q---
$D=(7-7y)^2$
---CORR---
$D=(7-7y)^2$<br>$D=(7-7y)(7-7y)$<br>$D= 7 \times 7 +  7 \times (-7y) +  (-7y) \times 7  + (-7y) \times (-7y) $<br>$D=49 - 49y - 49y + 49y^2$<br>$D=$ ${\color{#C5607A}\boldsymbol{49y^2-98y+49}}$

---Q---
$B = 2(2x-3)-x(2x-3)$
---CORR---
$B = 2(2x-3)-x(2x-3)$&thinsp;&thinsp;On remarque que $(2x-3)$ est un facteur commun.<br>$\phantom{ABC}={\color{#8B3C52}\boldsymbol{2}}{\color{#C5607A}\boldsymbol{(2x-3)}}-{\color{#8B3C52}\boldsymbol{x}}{\color{#C5607A}\boldsymbol{(2x-3)}}$<br>$\phantom{ABC}={\color{#C5607A}\boldsymbol{(2x-3)}}{\color{#8B3C52}\boldsymbol{(2-x)}}$<br>
---Q---
$C=(x-2)(5x+4)-(x-2)(x+2)$
---CORR---
$C=(x-2)(5x+4)-(x-2)(x+2)$<br>On remarque que $(x-2)$ est un facteur commun.<br>$C={\color{#C5607A}\boldsymbol{(x-2)}}{\color{#8B3C52}\boldsymbol{(5x+4)}}-{\color{#C5607A}\boldsymbol{(x-2)}}{\color{#8B3C52}\boldsymbol{(x+2)}}$<br>$C={\color{#C5607A}\boldsymbol{(x-2)}}{\color{#8B3C52}\boldsymbol{(5x+4-(x+2))}}$<br>$C={\color{#C5607A}\boldsymbol{(x-2)}}{\color{#8B3C52}\boldsymbol{(5x+4-x-2)}}$<br>$C={\color{#C5607A}\boldsymbol{(x-2)}}{\color{#8B3C52}\boldsymbol{(4x+2)}}$<br>
---Q---
$D=(3x-2)(2x+4)+(3x-2)(5x-3)$
---CORR---
$D=(3x-2)(2x+4)+(3x-2)(5x-3)$<br>On remarque que $(3x-2)$ est un facteur commun.<br>$D={\color{#C5607A}\boldsymbol{(3x-2)}}{\color{#8B3C52}\boldsymbol{(2x+4)}}+{\color{#C5607A}\boldsymbol{(3x-2)}}{\color{#8B3C52}\boldsymbol{(5x-3)}}$<br>$D={\color{#C5607A}\boldsymbol{(3x-2)}}{\color{#8B3C52}\boldsymbol{(2x+4+5x-3)}}$<br>$D={\color{#C5607A}\boldsymbol{(3x-2)}}{\color{#8B3C52}\boldsymbol{(7x+1)}}$<br>
