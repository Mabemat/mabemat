{{< qcm question="Deux triangles sont semblables si :" correct="1" >}}
- Ils ont le même périmètre
- Ils ont les mêmes angles
- Ils ont les mêmes côtés
- Ils sont rectangles
{{< /qcm >}}

{{< qcm question="Si deux triangles sont semblables de rapport $k = 2$, leurs périmètres sont dans le rapport :" correct="0" >}}
- $2$
- $4$
- $\sqrt{2}$
- $\frac{1}{2}$
{{< /qcm >}}

{{< qcm question="Leurs aires sont dans le rapport :" correct="1" >}}
- $2$
- $4$
- $8$
- $\frac{1}{4}$
{{< /qcm >}}

{{< qcm question="Le rapport de similitude est :" correct="2" >}}
- La différence des côtés
- La somme des côtés
- Le quotient de côtés homologues
- Le produit des périmètres
{{< /qcm >}}

{{< qcm question="Si $\triangle ABC \sim \triangle DEF$ avec $AB = 4$ et $DE = 6$, le rapport est :" correct="0" >}}
- $\frac{2}{3}$
- $\frac{3}{2}$
- $\frac{1}{2}$
- $2$
{{< /qcm >}}
