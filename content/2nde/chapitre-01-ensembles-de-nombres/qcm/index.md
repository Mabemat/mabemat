{{< qcm question="L'ensemble $\mathbb{Z}$ contient :" correct="1" >}}
- Uniquement les entiers positifs
- Tous les entiers relatifs
- Tous les décimaux
- Tous les rationnels
{{< /qcm >}}

{{< qcm question="Quel intervalle représente $-1 \leq x < 4$ ?" correct="1" >}}

- $[-1 ; 4[$
- $[-1 ; 4]$
- $]-1 ; 4]$
{{< /qcm >}}

{{< qcm question="$\sqrt{49}$ est un entier naturel. Lequel ?" correct="2" >}}
- $5$
- $6$
- $7$
- $8$
{{< /qcm >}}

{{< qcm question="Quelle expression est un irrationnel ?" correct="2" >}}
- $\frac{3}{7}$
- $0{,}333...$
- $\sqrt{2}$
- $\frac{22}{7}$
{{< /qcm >}}

{{< qcm question="$\frac{1}{4}$ appartient à :" correct="2" >}}
- $\mathbb{N}$
- $\mathbb{Z}$
- $\mathbb{D}$
- $\mathbb{Q} \setminus \mathbb{D}$
{{< /qcm >}}

{{< qcm question="$(a+b)^2$ est égal à :" correct="2" >}}

- $a^2+ab+b^2$
- $a^2+2ab+b^2$
- $a^2-2ab+b^2$
