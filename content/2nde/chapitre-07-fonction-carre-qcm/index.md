{{< qcm question="La fonction carré $f(x) = x^2$ est décroissante sur :" correct="1" >}}
- $[0 ; +\infty[$
- $]-\infty ; 0]$
- $\mathbb{R}$
- $]-\infty ; +\infty[$
{{< /qcm >}}

{{< qcm question="Le minimum de $f(x) = x^2$ est :" correct="2" >}}
- $1$
- $-\infty$
- $0$
- Il n'existe pas
{{< /qcm >}}

{{< qcm question="Résoudre $x^2 = 25$ :" correct="2" >}}
- $x = 5$
- $x = -5$
- $x = 5$ ou $x = -5$
- Pas de solution dans $\mathbb{R}$
{{< /qcm >}}

{{< qcm question="Résoudre $x^2 = -4$ dans $\mathbb{R}$ :" correct="3" >}}
- $x = 2$
- $x = -2$
- $x = 2$ ou $x = -2$
- Pas de solution réelle
{{< /qcm >}}

{{< qcm question="L'axe de symétrie de la parabole $y = x^2$ est :" correct="1" >}}
- L'axe des abscisses
- L'axe des ordonnées
- La droite $y = x$
- La droite $x = 1$
{{< /qcm >}}
