{{< qcm question="Lequel de ces nombres n’est PAS un entier naturel ?" correct="2" >}}
- $7$
- $0$
- $-3$
- $15$
{{< /qcm >}}

{{< qcm question="L’ensemble $\\mathbb{Z}$ contient :" correct="1" >}}
- Seulement les entiers positifs
- Les entiers positifs et négatifs
- Tous les rationnels
- Tous les réels
{{< /qcm >}}

{{< qcm question="$\\sqrt{2}$ appartient à :" correct="3" >}}
- $\\mathbb{N}$
- $\\mathbb{Z}$
- $\\mathbb{Q}$
- $\\mathbb{R} \\setminus \\mathbb{Q}$
{{< /qcm >}}

{{< qcm question="L’intervalle $[-2\\,;\\,5[$ contient :" correct="0" >}}
- $-2$ mais pas $5$
- $5$ mais pas $-2$
- Ni $-2$ ni $5$
- $-2$ et $5$
{{< /qcm >}}

{{< qcm question="$\\dfrac{1}{3}$ appartient à :" correct="1" >}}
- $\\mathbb{Z}$
- $\\mathbb{Q}$
- $\\mathbb{N}$
- $\\mathbb{R} \\setminus \\mathbb{Q}$
{{< /qcm >}}
