{{< qcm question="La fonction $f(x) = -2x + 5$ a pour pente :" correct="2" >}}
- $5$
- $2$
- $-2$
- $-5$
{{< /qcm >}}

{{< qcm question="L'ordonnée à l'origine de $f(x) = 3x - 4$ est :" correct="1" >}}
- $3$
- $-4$
- $4$
- $-3$
{{< /qcm >}}

{{< qcm question="Deux droites sont parallèles si elles ont :" correct="1" >}}
- La même ordonnée à l'origine
- La même pente
- Des pentes opposées
- Des pentes inverses
{{< /qcm >}}

{{< qcm question="La solution de $2x - 6 = 0$ est :" correct="2" >}}
- $x = 6$
- $x = -3$
- $x = 3$
- $x = 12$
{{< /qcm >}}

{{< qcm question="La droite $y = 2x - 1$ passe par le point :" correct="3" >}}
- $(0 ; 2)$
- $(1 ; 1)$
- $(1 ; 0)$
- $(0 ; -1)$
{{< /qcm >}}


{{< qcm question="L'équation $-3x + 9 \gt 0$ a pour solution :" correct="1" >}}
- $x \gt 3$
- $x \lt 3$
- $x \gt -3$
- $x \lt -3$
{{< /qcm >}}
