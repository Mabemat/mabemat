{{< qcm question="Pour trouver le maximum d'une fonction sur un intervalle, on cherche les valeurs de $x$ où :" correct="1" >}}
- $f(x) = 0$
- $f'(x) = 0$
- $f''(x) = 0$
- $f(x) > 0$
{{< /qcm >}}

{{< qcm question="Si $f'(x) < 0$ sur $[a;b]$, la fonction $f$ est :" correct="1" >}}
- Croissante
- Décroissante
- Nulle
- Positive
{{< /qcm >}}

{{< qcm question="On maximise $A(x) = -x^2 + 6x$ sur $[0;6]$. Le maximum est atteint pour :" correct="2" >}}
- $x = 0$
- $x = 6$
- $x = 3$
- $x = -3$
{{< /qcm >}}

{{< qcm question="Un problème d'optimisation se résout en :" correct="1" >}}
- Calculant $f(0)$
- Dérivant puis étudiant le signe de $f'$
- Calculant $f'(0)$
- Traçant la courbe
{{< /qcm >}}

{{< qcm question="Si $f'(a) = 0$ et $f'$ ne change pas de signe en $a$, alors $f(a)$ est :" correct="2" >}}
- Un maximum
- Un minimum
- Un point d'inflexion
- L'ordonnée à l'origine
{{< /qcm >}}
