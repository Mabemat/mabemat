{{< qcm question="$E(X)$ est :" correct="2" >}}
- La variance de $X$
- La valeur la plus probable
- La moyenne pondérée par les probabilités
- L'écart-type
{{< /qcm >}}

{{< qcm question="Si $X$ prend les valeurs $0$, $1$, $2$ avec probabilités $0{,}5$; $0{,}3$; $0{,}2$, alors $E(X) = $" correct="1" >}}
- $1$
- $0{,}7$
- $0{,}5$
- $1{,}5$
{{< /qcm >}}

{{< qcm question="$V(X) = E(X^2) - [E(X)]^2$. C'est :" correct="1" >}}
- L'espérance au carré
- La variance
- L'écart-type
- La loi de $X$
{{< /qcm >}}

{{< qcm question="$\sigma(X)$ (écart-type) est toujours :" correct="2" >}}
- Négatif
- Nul
- Positif ou nul
- Supérieur à $E(X)$
{{< /qcm >}}

{{< qcm question="$E(2X + 3) = $" correct="0" >}}
- $2E(X) + 3$
- $2E(X)$
- $E(X) + 3$
- $E(X)^2 + 3$
{{< /qcm >}}
