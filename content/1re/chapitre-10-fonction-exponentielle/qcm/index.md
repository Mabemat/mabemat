{{< qcm question="$e^0 = $" correct="1" >}}
- $0$
- $1$
- $e$
- Indéfini
{{< /qcm >}}

{{< qcm question="$e^{a+b} = $" correct="1" >}}
- $e^a + e^b$
- $e^a \times e^b$
- $e^{ab}$
- $(e^a)^b$
{{< /qcm >}}

{{< qcm question="La dérivée de $f(x) = e^x$ est :" correct="2" >}}
- $xe^{x-1}$
- $e^{x-1}$
- $e^x$
- $\frac{1}{e^x}$
{{< /qcm >}}

{{< qcm question="$e^x > 0$ pour :" correct="2" >}}
- $x > 0$ seulement
- $x > 1$ seulement
- Tout réel $x$
- $x \neq 0$
{{< /qcm >}}

{{< qcm question="La fonction exponentielle est :" correct="1" >}}
- Décroissante sur $\mathbb{R}$
- Croissante sur $\mathbb{R}$
- Constante
- Décroissante sur $\mathbb{R}^-$
{{< /qcm >}}

{{< qcm question="Résoudre $e^x = e^3$ :" correct="2" >}}
- $x = e^3$
- $x = 3e$
- $x = 3$
- $x = \ln 3$
{{< /qcm >}}
