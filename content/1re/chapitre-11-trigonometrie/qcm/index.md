- $1$
- $-1$
- $\frac{1}{2}$
{{< /qcm >}}

{{< qcm question="$\sin\left(\frac{\pi}{2}\right) = $" correct="2" >}}
- $0$
- $\frac{\sqrt{2}}{2}$
- $1$
- $\frac{1}{2}$
{{< /qcm >}}

{{< qcm question="$\cos^2(x) + \sin^2(x) = $" correct="2" >}}
- $0$
- $\cos(2x)$
- $1$
- $2$
{{< /qcm >}}

{{< qcm question="Les solutions de $\cos(x) = \frac{1}{2}$ sont :" correct="1" >}}
- $x = \frac{\pi}{3} + 2k\pi$ seulement
- $x = \pm\frac{\pi}{3} + 2k\pi$
- $x = \frac{\pi}{6} + 2k\pi$
- $x = \frac{\pi}{3} + k\pi$
{{< /qcm >}}

{{< qcm question="$\cos(\pi - x) = $" correct="1" >}}
- $\cos(x)$
- $-\cos(x)$
- $\sin(x)$
- $-\sin(x)$
{{< /qcm >}}

{{< qcm question="La période de $f(x) = \sin(x)$ est :" correct="1" >}}

- $\frac{\pi}{2}$
- $4\pi$
