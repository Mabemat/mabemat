{{< qcm question="La dérivée de $f(x) = x^3 - 2x + 5$ est :" correct="1" >}}
- $3x^2 + 5$
- $3x^2 - 2$
- $x^2 - 2$
- $3x^3 - 2$
{{< /qcm >}}

{{< qcm question="La dérivée de $f(x) = (2x+1)^3$ est :" correct="1" >}}
- $3(2x+1)^2$
- $6(2x+1)^2$
- $3 \times 2(2x+1)^2$
- $2(2x+1)^2$
{{< /qcm >}}

{{< qcm question="Si $f'(x) > 0$ sur $[a;b]$, alors $f$ est :" correct="1" >}}
- Décroissante sur $[a;b]$
- Croissante sur $[a;b]$
- Constante sur $[a;b]$
- Nulle sur $[a;b]$
{{< /qcm >}}

{{< qcm question="La dérivée de $f(x) = \frac{1}{x^2}$ est :" correct="1" >}}
- $\frac{2}{x^3}$
- $-\frac{2}{x^3}$
- $-2x$
- $\frac{1}{2x}$
{{< /qcm >}}

{{< qcm question="La dérivée du produit $u \times v$ est :" correct="1" >}}
- $u'v'$
- $u'v + uv'$
- $u'v - uv'$
- $\frac{u'v-uv'}{v^2}$
{{< /qcm >}}

{{< qcm question="Si $f'(a) = 0$ et $f'$ change de signe en $a$, alors $f(a)$ est :" correct="1" >}}
- Un point d'inflexion
- Un maximum ou minimum local
- La dérivée en $a$
- La valeur de la tangente
{{< /qcm >}}
