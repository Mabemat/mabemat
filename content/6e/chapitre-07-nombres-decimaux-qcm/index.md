{{< qcm correct="2" >}}
---Q---
Quelle est l'écriture décimale de $\dfrac{6375}{1000}$ ?

- $0{,}6375$
- $63{,}75$
- $6{,}375$
- Aucune réponse

---CORR---
On divise par $1000$, ce qui revient à déplacer la virgule de trois rangs vers la gauche :

$$\dfrac{6375}{1000} = 6{,}375.$$
{{< /qcm >}}


{{< qcm correct="1" >}}
---Q---
Laquelle de ces fractions est une **fraction décimale** ?

- $\dfrac{7}{3}$
- $\dfrac{14}{100}$
- $\dfrac{5}{12}$
- Aucune réponse

---CORR---
Une **fraction décimale** est une fraction dont le dénominateur est $10$, $100$, $1\,000$, etc.

Ici, seule : $\dfrac{14}{100}$ a un dénominateur décimal.
{{< /qcm >}}



{{< qcm correct="1" >}}
---Q---
Dans le nombre $851{,}67$, quel est le **chiffre des dixièmes** ?

- $7$
- $6$
- $5$
- Aucune réponse

---CORR---
Dans $851{,}67$ :
- $8$ est le chiffre des centaines,
- $5$ celui des dizaines,
- $1$ celui des unités,
- $6$ celui des dixièmes,
- $7$ celui des centièmes.

Le chiffre des dixièmes est donc $6$.
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Dans le nombre $851{,}67$, quel est le **nombre de dizaines** ?

- $6$
- $5$
- $85$
- Aucune réponse

---CORR---
Le nombre de dizaines, c’est le quotient de $851{,}67$ par $10$ pris entier :

$$851{,}67 = 85 \times 10 + 1{,}67.$$

Le **nombre de dizaines** est donc $85$ (à ne pas confondre avec le chiffre des dizaines, qui est $5$).
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Range dans l'ordre croissant : $3{,}7$ ; $3{,}07$ ; $3{,}007$.

- $3{,}7 < 3{,}07 < 3{,}007$
- $3{,}07 < 3{,}007 < 3{,}7$
- $3{,}007 < 3{,}07 < 3{,}7$
- Aucune réponse

---CORR---
On compare les nombres en complétant éventuellement avec des zéros :

$$3{,}7 = 3{,}700,\quad 3{,}07 = 3{,}070,\quad 3{,}007 = 3{,}007.$$

On obtient :

$$3{,}007 < 3{,}07 < 3{,}7.$$
{{< /qcm >}}

{{< qcm correct="0" >}}
---Q---
Arrondis $7{,}348$ au **dixième**.

- $7{,}3$
- $7{,}4$
- $7{,}35$
- Aucune réponse

---CORR---
On regarde le chiffre des centièmes : c’est $4$.

Comme $4 < 5$, on n’augmente pas le chiffre des dixièmes :

$$7{,}348 \approx 7{,}3.$$
{{< /qcm >}}

{{< qcm correct="1" >}}
---Q---
$91 \div 33 \approx 2{,}757\ 576$.  
Si on arrondit au **centième**, on obtient :

- $2{,}7$
- $2{,}76$
- $2{,}75$
- Aucune réponse

---CORR---
On regarde le chiffre des millièmes : c’est **7**, donc on arrondit le centième vers le haut :

$$2{,}757\ 576 \approx 2{,}76$$
{{< /qcm >}}



{{< qcm correct="1" >}}
---Q---
Le nombre **36 %** est égal à :

- $3{,}6$
- $\dfrac{36}{100}$
- $36$
- Aucune réponse

---CORR---
Par définition :

$36$ % $ = \dfrac{36}{100} = 0{,}36.$

Donc l’écriture fractionnaire est $\dfrac{36}{100}$.
{{< /qcm >}}
