{{< qcm correct="2" >}}
---Q---
$56$ est **divisible par 7** car :

- $56 + 7 = 63$
- $56 - 7 = 410$
- $56 = 7 \times 8$
- $56 > 7$

---CORR---
Un entier $a$ est divisible par $b$ s'il existe un entier $q$ tel que $a = b \times q$.

$56 = 7 \times 8$ → **56 est divisible par 7**, 56 est un multiple de 7, et 7 est un diviseur de 56.
{{< /qcm >}}

{{< qcm correct="1" >}}
---Q---
Lequel de ces nombres est divisible par **3** ?

- 154
- 231
- 253
- Aucune réponse

---CORR---
Critère de divisibilité par 3 : la **somme des chiffres** doit être un multiple de 3.

- $1+2+4 = 7$ ✗
- $\mathbf{2+3+1 = 6}$ ✓ (multiple de 3)
- $2+5+3 = 15$ ✗
- $3+4+1 = 8$ ✗
{{< /qcm >}}

{{< qcm correct="0" >}}
---Q---
Lequel de ces nombres est divisible par **4** ?

- 132
- 134
- 138
- Aucune réponse

---CORR---
Critère de divisibilité par 4 : les **deux derniers chiffres** forment un multiple de 4.

- $32 = 4 \times 8$ ✓ → **132** est divisible par 4
- $34 \div 4 = 8{,}5$ ✗
- $38 \div 4 = 10{,}5$ ✗
- $30 \div 4 = 7{,}5$ ✗
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Effectue la division euclidienne de **47 par 6**. Quel est le reste ?

- 6
- 7
- 5
- Aucune réponse

---CORR---
$47 = 6 \times 7 + 5$

Le **quotient** est 7 et le **reste** est **5** ($0 \leq 5 < 6$ ✓).
{{< /qcm >}}

{{< qcm correct="1" >}}
---Q---
Dans la division euclidienne de $a$ par $b$, on obtient le quotient $q$ et le reste $r$. Laquelle de ces égalités est vraie ?

- $a = b + q + r$
- $a = b \times q + r$
- $a = b \times q \times r$
- $a = b + q \times r$

---CORR---
La relation de la **division euclidienne** est :

$$a = b \times q + r \quad \text{avec} \quad 0 \leq r < b$$
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Lequel de ces nombres est divisible à la fois par **2** et par **3** ?

- 14
- 21
- 42
- Aucune réponse

---CORR---
- Divisible par 2 : chiffre des unités pair.
- Divisible par 3 : somme des chiffres multiple de 3.

Pour **42** : chiffre des unités = 2 (pair ✓) et $4+2 = 6$ (multiple de 3 ✓).
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Lequel de ces nombres est divisible par **5** ?

- 23
- 34
- 35
- Aucune réponse

---CORR---
Critère de divisibilité par 5 : le **chiffre des unités** doit être **0 ou 5**.

**35** se termine par 5 → il est divisible par 5.
{{< /qcm >}}
