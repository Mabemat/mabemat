{{< qcm correct="2" >}}
---Q---
Deux figures sont **symétriques par rapport à une droite** $(d)$ si :

- On a fait un demi-tour.
- Elles sont dans le même sens
- Elles se superposent par **pliage** le long de $(d)$
- Aucune réponse

---CORR---
La définition de la **symétrie axiale** : deux figures sont symétriques par rapport à $(d)$ si elles se **superposent exactement par pliage** le long de $(d)$.
{{< /qcm >}}

{{< qcm correct="1" >}}
---Q---
Le symétrique d'un point $A$ (n'appartenant pas à l'axe $(d)$) est le point $A'$ tel que :

- $A$ est sur $(d)$
- $(d)$ est la **médiatrice** du segment $[AA']$
- $(d)$ est parallèle à $[AA']$
- Aucune réponse

---CORR---
Si $A$ n'est pas sur l'axe $(d)$, son symétrique $A'$ est tel que $(d)$ est la **médiatrice de $[AA']$** : $(d)$ est perpendiculaire à $[AA']$ et passe par le milieu de $[AA']$.
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Si un point $C$ appartient à l'axe de symétrie $(d)$, alors son symétrique est :

- N'importe quel point de $(d)$
- Il n'a pas de symétrique
- Lui-même
- Aucune réponse

---CORR---
Si un point est **sur l'axe**, il est son **propre symétrique** (le pliage le laisse à la même place).
{{< /qcm >}}

{{< qcm correct="3" >}}
---Q---
Le symétrique du segment $[AB]$ est le segment $[A'B']$. Que peut-on affirmer ?

- $[AB] = [A'B']$
- $AB = 2 \times A'B'$
- $A'B' = 0$
- Aucune réponse

---CORR---
La symétrie axiale **conserve les longueurs** : le symétrique d'un segment a la même longueur.

$$AB = A'B'$$
{{< /qcm >}}

{{< qcm correct="1" >}}
---Q---
La symétrie axiale conserve :

- Uniquement les longueurs
- Les longueurs **et** les mesures d'angles
- Uniquement les angles
- Aucune réponse

---CORR---
La symétrie axiale conserve :
- Les **longueurs** des segments
- Les **mesures d'angles**
- L'**alignement** des points
{{< /qcm >}}

{{< qcm correct="2" >}}
---Q---
Combien d'axes de symétrie possède un **rectangle non carré** ?

- 0
- 1
- 2
- Aucune réponse

---CORR---
Un rectangle (non carré) possède **2 axes de symétrie** : les droites passant par les milieux des côtés opposés (un axe « horizontal » et un axe « vertical »).
{{< /qcm >}}

{{< qcm correct="3" >}}
---Q---
Combien d'axes de symétrie possède un **triangle équilatéral** ?

- 0
- 1
- 2
- Aucune réponse

---CORR---
Un triangle équilatéral possède **3 axes de symétrie** : chaque axe passe par un sommet et le milieu du côté opposé.
{{< /qcm >}}
